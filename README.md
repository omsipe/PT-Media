# PT-Media Website

Statische Landing Page für PT-Media — Veranstaltungstechnik & App Development.

## Deployment

Die Seite ist eine einzelne `index.html` mit zwei Bilddateien — keine Build-Tools nötig.

### GitHub Pages

1. Repository auf GitHub erstellen (z. B. `pt-media-website`).
2. Alle Dateien aus diesem Ordner ins Repo pushen.
3. In den Repo-Settings → **Pages** → Source: `main` / root.
4. Seite ist nach 1-2 Min unter `https://DEIN-USER.github.io/pt-media-website/` erreichbar.

## Anpassen

- GitHub-Links (`DEIN-USER`) in `index.html` durch deinen Username ersetzen.
- Tools/Apps in der Sektion `apps-grid` ergänzen oder entfernen.

## Dateien

- `index.html` — die Webseite
- `PT-Media-Wordmark-v2.png` — Wortmarke (Hero & Nav)
- `PT-Media-Logo.png` — Logo (für Social, Favicon, etc.)
