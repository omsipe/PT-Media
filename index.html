Soundboard Stable
v1.0.0 · Windows

Pad-basiertes Soundboard mit Trim-Punkten, Fades und Companion-Anbindung — steuerbar per Stream Deck.

  • MP3/WAV mit Drag & Drop
  • Trim Start/End pro Pad
  • Fade In/Out mit Auto-Fade-Ende
  • Performance-Mode für Live-Einsatz
  • Companion-Bridge integriert
↓ Download GitHub
Companion-Modul Stable
v1.0.0 · Cross-Platform

Bitfocus-Companion-Modul für das Soundboard. Pads als Drag-and-drop-Presets mit Live-Status und Restzeit auf den Stream-Deck-Buttons.

  • Pad-Presets in Companion
  • Live-Variablen (Name, Restzeit, Status)
  • Visual Feedback: Button leuchtet beim Spielen
  • Trigger / Toggle / Stop pro Pad
